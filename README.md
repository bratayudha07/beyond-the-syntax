# Beyond The Syntax

**Memahami Python di Balik Syntax**

Hai! 👋  

Selamat datang di **Beyond The Syntax**.  

Repo ini dibuat karena saya ingin belajar Python dengan cara yang lebih dalam. Bukan cuma syntax dan trik coding, tapi memahami **mental model**-nya, cara kerjanya di balik layar, dan filosofi yang bikin Python terasa “Python banget”.

### Status Saat Ini

> ⚠️ Proyek ini masih sangat awal dan terus saya kembangkan pelan-pelan.

Seluruh isi repo ini **dibuat dengan bantuan penuh Artificial Intelligence**, lalu saya edit dan arahkan sesuai visi saya.
